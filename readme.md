### 用于命令行的英语词典

#### 功能
- 英文到中文的翻译查询
- 类似多邻国的单词记忆练习

##### 配置

配置文件默认位置：`~/.config/idict/idict.yaml`

- `StoragePath`: 存储位置，默认：`~/.local/share/idict`
- `LoopNum`: 一组练习的单词数量，这一组单词不断循环出现，直到拼写正确
- `RestudyInterval`: 一个单词的连续正确次数，与复习时间间隔，以小时为单位

    例子

        ```
            RestudyInterval:
              2:0
              4:24
              6:96
              15:-1
        ```

    这代表:

        2 < 连续正确输入次数          会立即重复
        2 <= 连续正确输入次数 < 4     会在 1 小时后重复
        4 <= 连续正确输入次数 < 6     会在 24 小时后重复
        6 <= 连续正确输入次数 < 15    会在 96 小时后重复
        15 <= 连续正确输入次数        不再重复

- `FfplayPath`: 设置 ffplay 可以启用单词发音
- `FfplayArgs`: ffplay 的参数
